# Algorithms Visualization

An interactive webpage that shows how sorting algorithms work step by step.

## About This Project

When I first started learning about data structures and algorithms, one of my biggest pain points was trying to mentally picture how data moves through each algorithm.
Some algorithms are straightforward to follow, but others especially those involving recursion are much harder to trace in your head.

That got me thinking: "It would be so much easier if I could actually see the algorithm in action."

So I built this interactive visualizer. Instead of imagining the steps, you can watch them unfold in real time and follow along at your own pace.

On top of that, I'm currently learning Javascript, that's why I decided to start this project where I can practice my javascript skills alongside my understanding of the algorithms at the same time.

Check it out here → https://minjk25.github.io/algorithms-visualization/
